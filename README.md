# Quiz-site (Demo)

Trang web trắc nghiệm tĩnh (GitHub Pages) với dữ liệu từ CSV.
Frontend gửi kết quả tới Google Apps Script để lưu vào Google Sheets / Drive.

Các bước:
1. Cập nhật file `questions.csv`.
2. Triển khai Google Apps Script (xem hướng dẫn trong tài liệu).
3. Cập nhật `GAS_ENDPOINT` trong `app.js`.
4. Push repo lên GitHub và bật GitHub Pages.
